# Information Architecture

## 页面目标
构建清晰、可扩展、低耦合的内部工作台结构。

## 页面层级
- 一级导航：Dashboard / Projects / Tasks / Versions / Docs Index / Change Log / References / Settings
- 二级页面：Project Detail (`/projects/[projectId]`)

## 模块职责
- Dashboard：全局概览
- Projects：项目边界管理
- Tasks：任务推进管理
- Versions：版本链治理
- Docs Index：文档入口治理
- Change Log：变更追踪
- References：候选引用审核
- Settings：治理规则展示
